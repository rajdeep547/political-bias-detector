# 📰 Political Bias Detector

A fine-tuned BERT model for detecting political bias in news articles.

## Features
- Classifies articles as **Left**, **Neutral**, or **Right**
- Shows confidence scores for each prediction
- Easy-to-use web interface

## Model Details
- **Base Model**: BERT-base-uncased
- **Dataset**: TriLabel Political News (10,000 samples)
- **Accuracy**: ~85%
